# Bed Nest Hostel Website

A modern, professional hostel booking website built with pure HTML, CSS, Font Awesome, and Outfit font. No JavaScript frameworks, no gradients - just clean, outstanding design.

## 🏨 Project Overview

Bed Nest is a premium budget hostel website featuring:
- **Complete booking experience** from browsing to confirmation
- **Mobile-first responsive design** that works on all devices
- **Professional trust signals** with policies, testimonials, and contact info
- **Clean, modern aesthetic** using solid colors and professional typography

## 🎨 Design Features

### Visual Design
- **No gradients** - Clean flat design with solid colors
- **Outfit font family** - Modern, professional typography
- **Font Awesome icons** - Consistent iconography throughout
- **Responsive grid layouts** - Perfect on mobile, tablet, and desktop
- **Professional color palette** - Teal brand color with warm accents

### Trust & Credibility
- ⭐ Customer testimonials with ratings
- 🛡️ Security badges and trust indicators  
- 📱 24/7 contact information
- 📋 Clear policies (cancellation, house rules, etc.)
- 🏆 Awards and certifications section
- 💳 Transparent pricing with breakdown

## 📄 Pages Included

| Page | Purpose | Key Features |
|------|---------|--------------|
| **Homepage** (`index.html`) | Landing & overview | Hero, room highlights, amenities, testimonials |
| **Rooms** (`rooms.html`) | Room comparison | Detailed specs, comparison table, policies |
| **Gallery** (`gallery.html`) | Visual showcase | Photo grid, categories, virtual tour CTA |
| **About** (`about.html`) | Brand story | Company story, values, team, statistics |
| **Contact** (`contact.html`) | Get in touch | Contact form, location, FAQ, emergency info |
| **Booking** (`booking.html`) | Make reservation | Comprehensive form with live price summary |
| **Thank You** (`thank-you.html`) | Confirmation | Booking confirmation with next steps |
